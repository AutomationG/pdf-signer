## Prerequisites
- Node.js (22.12+)
- npm or yarn

## Quick start
1. Open terminal and go to the client folder:
  - cd pdf-signer/client
2. Install dependencies:
  - npm install
  - or yarn
4. Run in development:
  - npm run dev 
  - or yarn dev
  - open app in browser
5. Build for production:
  - npm run build
